PyConTW 徵求講題及投稿方式

主題範圍

2013 台灣 Python 年會徵求各式領域應用 Python 的案例分析或經驗心得，以下列舉僅是可能的主題範例，不代表只侷限於這些主題：

  • Web Programming (e.g Django, Pyramid, Web2Py, TurboGears)
  • Cloud Computing
  • Scientific Computing - Startups, Business, and Education
  • Multimedia, Animation, and Graphics
  • Game Programming
  • GUI Programming
  • Hardware/Embedded System Design
  • System Administration and Security
  • Network Programming
  • Agile Development and Project Best Practices
  • Packaging Issues
  • Programming Tools
  • Python Libraries and Extensions
  • Python Implementations: IronPython, Jython, PyPy and Stackless.

提案填寫及投影檔案上傳

在真正上傳提案之前，必須先到 http://tw.pycon.org/zh/2013/speaker/ 網址「註冊」帳號，登入帳號後才可以填寫提案內容，並上傳投影檔案。

提案或演講的主要內容，限用中文或英文進行說明，上傳時並請選定演講的主要語言。一個講題提案，可以由多位講者共同發表，但必須由主要講者 (primary presenter) 執行上傳工作，並至少提供下列的提案資訊，以便議程委員於審稿期間，提供適當的協助，並擬定最佳的議程安排：

  • 講題標題
  • 演講類型 (一般演講 或 閃電秀)
  • 主要語言 (中文 或 英文)
  • 主題類型
  • 聽眾對象
  • 講題描述 (至少150字)
  • 講題摘要 (最多200字)
  • 同意你的演講進行錄影並公開在 PyCon Taiwan 網站
  • 同意你的投影檔案在 PyCon Taiwan 網站公開下載
  • 附註說明 (例如你的工作介紹或使用 Python 的歷史)

請參考提案範本以了解上傳時需要填寫的欄位資訊。

投影檔案/論文摘要 上傳方式

提案審核通過後，就可以上傳完整的投影檔案，或是論文摘要。

投影檔案上傳

請善用投影檔案的註釋欄位，盡量提供相關的解說訊息，以便聽眾或讀者更容易理解內容。並從 http://tw.pycon.org/zh/2013/speaker/ 網址進行上傳動作。

論文摘要上傳

講者如果想要上傳短篇論文 (short paper) 或是延伸摘要 (extended abstract)，本屆 PyCon Taiwan 將協助在 The Python Papers Monograph 建立研討會論文集 (unrefereed volume 不包括投影檔案)，請上傳 10 頁以內的論文檔案，並使用 PDF 或 LibreOffice Write 格式的檔案，從 http://tw.pycon.org/zh/2013/speaker/ 網址進行上傳動作。

對於上傳論文摘要的講者，PyCon Taiwan 會後將另行邀稿，上傳完整的論文 (包括摘要、簡介、內文、結論等)，經過同儕評審後，在 The Python Papers 發表出版。
