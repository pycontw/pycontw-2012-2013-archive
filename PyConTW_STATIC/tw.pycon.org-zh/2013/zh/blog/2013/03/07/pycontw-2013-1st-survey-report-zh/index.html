PyConTW 2013 第一階段問卷結果搶先報

首先感謝各位朋友填寫這次 PyConTW 2013 的問卷！一個多月以來，已經有百來位朋友願意給予我們回饋。除了持續請各位朋友分享此問卷給身邊的人作答以外，我們將在此摘要問卷的回答狀況以供參考：

  • 大家最喜歡聽的前三大演講主題依序為 Web（68%）、雲端（59%）、GUI（49%），緊接著在後的是 Mobile、資料庫與系統管理。有意思的是，對創業分享主題有興趣的朋友佔 43%，對科學有興趣的佔 29%，都是極具特色的主題和聽眾族群。由此可知，在台灣 Python 的使用者可能以 Web 應用為大宗，且有很高的機率都正在創業中或對創業有高度興趣！

  • 今年 Python Tutorial 的課程得到了不錯的反應，已有 24 位朋友表達一定會去參加講師林信良 (良葛格) 的課程，請有興趣者持續關注 先修課程頁面 的消息。

  • 調查各領域 Python 人才的需求：同樣地，以 Web（54%） 開發者的需求為最大宗，緊接在後的需求是 Database 與 General Purpose 以 34% 並列。事實上，無論是 GUI、Mobile、桌面程式、Python+數學、Python+設計美工、Python+硬體控制、Python+管理，這些人才需求的選項皆有人填寫。

  • 大家對於各地社群的看法，最多的問題就是詢問什麼時候台灣南部（台南、嘉義、雲林）會辦理 Python 的聚會。（對於此問題，PyConTW 的志工 Joe 表示，公司年中從台北搬遷回台南後，他會嘗試於台南尋找場地舉辦聚會，等不及的朋友不妨找個咖啡廳，先舉辦小型的聚會試試水溫。）

依據問卷的結果，我們預計執行以下的計畫：

  • 我們將持續依（但不侷限於）各主題，徵求國內外的講者共同參與這次 PyConTW 2013 的盛會。
  • 我們將在兩天的會議之中，試著規劃一些活動，促進有 Python 人才需求者與 Pythoner 之間的互動。
  • 對於也有興趣加入志工的朋友，我們將逐一與您聯絡。若您未留下聯絡資料，也不妨主動與我們 聯絡。

從問卷的結果觀之，最被關注的主題與需要的人才，皆是以 Web、Database 方面為主，合理推估是因問卷觸及到的早期作答者大多是 Web 開發人員，比較容易得知 PyConTW 2013 的資訊。因此，若是您的身邊有使用 Python 的科學怪人、嵌入蟻式人、IC 人 … 或是有興趣的人，請您務必將問卷給予他們作答，順便問看看有沒有興趣 投個演講 至 PyConTW 2013！再次感謝各位朋友們的協助！

（本文由 PyConTW 2013 志工 Joe 撰寫於 2013.3.5）
