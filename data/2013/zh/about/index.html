PyCon Taiwan 簡介

PyCon Taiwan (PyConTW) 是台灣 Python 社群的年會，獻於社群、取於社群、屬於社群。本研討會著重於 Python 技術與其廣泛的應用。我們召聚台灣的 Python 程式人員來參與這個會議，在規畫完善的環境中分享知識、討論想法。希望在這樣的氣氛裡刺激大家的靈感，並鼓動技術上的合作。歡迎所有對 Python 有興趣的事業團體一起來加入這個社群。

PyCon Taiwan 有哪些玩意（內容）：

  • Keynote 主題演講： 多樣主題的演講內容讓會眾盡情的享受 Python 的魅力。
  • Tutorial： 對 Python 很陌生嗎？不打緊！會前教學將輕鬆帶你踏入 Python 的世界。
  • 各式攤位： 想知道誰對 Python 有興趣嗎？到處逛逛發現知音吧！
  • Lightning Talks： 敬請期待短短5分鐘的演講如何讓整場聚會 high 到最高點吧！
